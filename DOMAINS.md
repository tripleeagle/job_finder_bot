# Domain candidates

Grouped by feel. Check availability before falling in love with one.

## Signal / dispatch theme (matches the current site design)

These tie to the product idea "we listen, you receive." They line up with the visual direction in `public/index.html`.

- **jobsignal.cz** — clean, English-readable, "signal" reads in CZ
- **jobsignal.io** — premium TLD if .cz is gone
- **praceradar.cz** — Czech-native, evocative
- **sonarjobs.cz** — distinctive, ties to "we hear it first"
- **jobsonar.cz**
- **pingjobs.cz** — short, modern
- **dispatch.jobs** — the .jobs TLD costs more but is on-brand
- **jobwire.cz**
- **jobcast.cz**
- **vlnaprace.cz** — "wave of jobs" in Czech, radio metaphor

## Czech-native, no metaphor needed

These read as Czech-first. Best for SEO in CZ and trust signals.

- **nadosah.cz** — "within reach"
- **nahledni.cz** — "peek/glance"
- **zachytprace.cz** — "catch work"
- **vhodnaprace.cz** — "fitting work"
- **najdiprace.cz** — "find work" (probably taken)
- **mojeprace.cz** — "my work" (probably taken)
- **pracenadalku.cz** — "work at a distance" (skews remote)

## Bare / brandable

Short, no built-in meaning. Easier to grow into a different product later.

- **levr.cz** / **levr.io**
- **murmur.jobs**
- **hoverjobs.cz**
- **driftjobs.cz**
- **tunejobs.cz**
- **lensjobs.cz**

## My top 5 (ranked)

1. **jobsignal.cz** — matches the design language, scales to .io if cz is taken
2. **praceradar.cz** — most Czech-native of the signal-themed options
3. **nadosah.cz** — clean Czech, no metaphor needed, easy to remember
4. **pingjobs.cz** — short, modern, the verb "ping" works in CZ
5. **dispatch.jobs** — most premium, on-theme, costs more

## Quick availability check

```bash
# Whois via nic.cz for .cz domains
whois jobsignal.cz
whois praceradar.cz
whois nadosah.cz

# Or just visit https://www.nic.cz/whois/?type=domain
```

For .io / .com / .jobs, check namecheap or Cloudflare Registrar.

## Notes

- The site copy and wordmark in `public/index.html` currently use **"jobsignal"** as a placeholder. To swap: search `public/index.html` for `jobsignal` and the `<title>` tag.
- The design direction (sodium-amber signal accent, dispatcher console feel) suits the *signal/dispatch* group above better than the Czech-native or brandable groups. If you pick from those instead, the visuals would want a small refresh — happy to redo.
